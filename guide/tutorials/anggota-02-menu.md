# Anggota 02 — Menu dashboard

## Tujuan
Anggota mengenal beranda, pintasan, dan menu penting.

## Akun demo
Anggota aktif (sudah diverifikasi).

## Storyboard
1. Login → land di `/dashboard`.
2. Tunjukkan header: Panduan, Pesan, Notifikasi.
3. Tunjukkan kartu anggota + checklist kelengkapan.
4. Buka **Aksi cepat**: Absensi, Iuran, Event, Pesan, Sabuk, Dokumen/Materi.
5. Buka sheet **Lainnya**: Store, Piagam, Pelatihan, Pindah Dojo, Riwayat.
6. Buka tab bawah **Profil**.
7. Buka ikon **Panduan** → `/dashboard/panduan`.

## Narasi VO
"Setelah masuk, beranda menampilkan kartu anggota dan pintasan fitur. Gunakan aksi cepat untuk absensi, iuran, dan kegiatan. Menu Lainnya berisi fitur tambahan. Profil ada di tab bawah. Panduan lengkap bisa dibuka kapan saja dari ikon Panduan."

## Teks on-screen
- Beranda anggota
- Aksi cepat
- Panduan

## Checklist rekaman
- [ ] Header sticky terlihat
- [ ] Grid aksi cepat
- [ ] Profil + panduan

## Jangan tampilkan
Notifikasi berisi data sensitif anggota lain; nominal UKT jika kebetulan muncul di kartu (seharusnya tidak).
