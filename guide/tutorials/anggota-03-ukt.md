# Anggota 03 — Daftar UKT mandiri

## Tujuan
Anggota mendaftar UKT dari kartu Status UKT, memahami syarat, dan konfirmasi bayar offline.

## Akun demo
Anggota aktif, periode UKT buka, syarat terpenuhi (atau sengaja kurang satu syarat untuk demo modal gate).

## Storyboard
1. Beranda → kartu **Status UKT** (atau Prestasi → Sabuk).
2. Jika syarat kurang: klik **Daftar UKT sekarang** → tampilkan modal checklist + CTA (iuran/dokumen/absensi).
3. Jika syarat OK: **Daftar UKT sekarang** → status **Menunggu Terima Ranting**.
4. Jelaskan bayar **offline ke ranting** (tanpa angka Rp).
5. Klik **Konfirmasi sudah bayar** → status **Menunggu Konfirmasi Ranting**.
6. Ringkas: ranting Terima → cabang Verifikasi → menunggu ujian.

## Narasi VO
"Daftar UKT dari kartu Status UKT di beranda. Pastikan iuran, dokumen Akte dan BPJS, serta absensi memenuhi syarat. Setelah daftar, bayar langsung ke ranting Anda, lalu konfirmasi sudah bayar di aplikasi. Nominal biaya dikonfirmasi pengurus ranting — tidak ditampilkan di dashboard anggota."

## Teks on-screen
- Daftar UKT sekarang
- Syarat UKT
- Konfirmasi sudah bayar
- Tanpa nominal di layar anggota

## Checklist rekaman
- [ ] Kartu Status UKT
- [ ] Gate syarat (opsional) atau daftar sukses
- [ ] Konfirmasi bayar
- [ ] **Tidak ada angka Rp biaya UKT** di frame mana pun

## Jangan tampilkan
Nominal biaya UKT; tagihan UKT di menu Iuran (sudah difilter).
